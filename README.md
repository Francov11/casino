## Comenzando 

Haz un clon del repositorio

```
git clone https://github.com/Francov11/casino
```

Instala las dependencias

```
npm install
```

### Instrucciones 📄

**1 - Ejecución**

* Local

Para iniciar el proyecto de manera local, tienes que usar `npm run start`
